Question: The tree will cost $90 to plant. Each year it will grow 7 lemons, which he can sell for $1.5 each. It costs $3 a year to water and feed the tree. How many years will it take before he starts earning money on the lemon tree?
Solution:
def solution():
    """The tree will cost $90 to plant. Each year it will grow 7 lemons, which he can sell for $1.5 each. It costs $3 a year to water and feed the tree. How many years will it take before he starts earning money on the lemon tree?"""
    # Define the initial cost to plant the tree and the annual maintenance cost
    INITIAL_COST = 90
    ANNUAL_MAINTENANCE_COST = 3

    # Define the price per lemon and the number of lemons produced each year
    LEMON_PRICE = 1.5
    LEMONS_PER_YEAR = 7

    # Initialize the year counter and the total earnings
    years = 0
    total_earnings = 0

    # Calculate the total cost of maintaining the tree for each year
    annual_cost = ANNUAL_MAINTENANCE_COST

    # Calculate the earnings from selling lemons each year, until the total earnings exceed the initial cost
    while total_earnings < INITIAL_COST:
        # Increment the year counter
        years += 1
        
        # Calculate the earnings from selling lemons
        lemon_earnings = LEMON_PRICE * LEMONS_PER_YEAR
        
        # Add the earnings to the total
        total_earnings += lemon_earnings
        
        # delete the annual maintenance cost 
        total_earnings -= ANNUAL_MAINTENANCE_COST
        
    # Display the number of years it took to start earning money
    result = years + 1
    return result

Question: In a dance class of 20 students, 20% enrolled in contemporary dance, 25% of the remaining enrolled in jazz dance, and the rest enrolled in hip-hop dance. What percentage of the entire students enrolled in hip-hop dance?
Solution:
def solution():
    """In a dance class of 20 students, 20% enrolled in contemporary dance, 25% of the remaining enrolled in jazz dance, and the rest enrolled in hip-hop dance. What percentage of the entire students enrolled in hip-hop dance?"""
    # Define the total number of students
    total_students = 20

    # Calculate the number of students enrolled in contemporary dance
    contemporary_students = total_students * 0.2

    # Calculate the number of remaining students
    remaining_students = total_students - contemporary_students

    # Calculate the number of students enrolled in jazz dance
    jazz_students = remaining_students * 0.25

    # Calculate the number of students enrolled in hip-hop dance
    hiphop_students = total_students - contemporary_students - jazz_students

    # Calculate the percentage of students enrolled in hip-hop dance
    hiphop_percentage = (hiphop_students / total_students) * 100

    # return the result
    result = hiphop_percentage
    return result
